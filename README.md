# University Analytics Dashboard

A lightweight, memory-efficient Python command-line application designed to process, analyze, and filter student academic records. This project focuses on pure programmatic logic, utilizing structured data types (Lists and Dictionaries) to manage state dynamically without external dependencies or file operations.

## Key Features
- **Dynamic Data Management:** In-memory storage utilizing nested structures (lists of dictionaries).
- **Input Validation:** Idiomatic error/boundary checking ensuring data integrity (e.g., grade verification between 0 and 20).
- **Algorithmic Analysis:** Custom mathematical implementations for computing grade averages, extracting maximum thresholds, and tracking performance milestones.
- **Data Filtering:** Conditional extraction algorithms to dynamically filter and report underperforming entities.

## Core Concepts Applied
- Control Flow & Iteration (`while` loops, `for-each` dictionary traversal)
- Conditional Branching (`if-elif-else`)
- Modular Programming (Custom Python Functions)
- String Formatting (f-strings with float precision capping)
